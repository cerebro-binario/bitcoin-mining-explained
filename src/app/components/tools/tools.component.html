Ferramentas

Um hash é como uma impressão digital para informações.

Ele pega um pedaço de dados (como um texto, um número ou até mesmo um arquivo inteiro) e transforma isso em uma sequência única de letras e números. Essa sequência sempre terá o mesmo tamanho, independentemente do tamanho do dado original.

Hexadecimal: é o formato que o computador "vê o conteúdo" da informação.

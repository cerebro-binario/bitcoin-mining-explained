Para comparação: mesmo com {{ formatBigNumber(hashesPerSecond) }} H/s, você testaria apenas {{ formatBigNumber(currentAttempt) }} chaves em {{ formatTime(elapsedTime) }}.

Por que é Impossível?

Espaço de Chaves: Uma chave privada Bitcoin tem 256 bits, resultando em 2²⁵⁶ possibilidades (aproximadamente 10⁷⁷).
Tempo Necessário: Mesmo com 1 quintilhão de hashes por segundo, levaria bilhões de anos para testar todas as possibilidades.
Energia: A energia necessária seria maior que toda a energia disponível no universo conhecido.
Segurança: Esta é a base da segurança criptográfica do Bitcoin - a impossibilidade computacional de quebrar chaves privadas.
